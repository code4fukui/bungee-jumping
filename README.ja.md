# VRバンジージャンプ

VRバンジージャンプを体験できるWebアプリケーションです。

## デモ
https://code4fukui.github.io/bungee-jumping/

## 機能
- バンジージャンプのアニメーション表示
- オキュラスタッチコントローラーによる操作
- 背景画像の切り替え

## 使い方
ブラウザで上記デモページを開くと、自動的にカウントダウンが始まりバンジージャンプのアニメーションが開始します。
オキュラスタッチコントローラーのトリガーやグリップボタンを押すと、リセットされてジャンプを再開できます。
また、Bボタンを押すと背景画像が切り替わります。
