# バンジージャンプ VR

A-Frameで構築された没入型WebXRバンジージャンプシミュレーション。物理演算ベースのモーションとインタラクティブな操作を備えています。

## デモ

VRブラウザで実際に体験してみてください: **https://github.com/code4fukui/bungee-jumping

## 特徴

- **物理演算ベースのモーション:** 重力、弾性力（バンジーコード）、空気抵抗を用いてジャンプをシミュレーションします。
- **没入感のある一人称視点:** 落下とバウンドの連続的な動きをリアルに再現します。
- **インタラクティブなスカイボックス:** VRコントローラーを使用して360度の背景環境を変更できます。
- **カウントダウンタイマー:** ジャンプ開始前に3秒間のカウントダウンが行われます。

## 必要条件

- コントローラー付きのVRヘッドセット（例: Oculus Quest/Rift）
- WebXR対応ブラウザ

## 体験方法

1. VRヘッドセットのブラウザで[デモURL](https://code4fukui.github.io/bungee-jumping/)を開きます。
2. 最初の3秒間のカウントダウン後、シミュレーションが自動的に開始されます。
3. コントローラーを使用してシーンを操作します。

### コントロール（Oculus Touch）

- **ジャンプのリセット:** 左右どちらかのコントローラーの**トリガー**または**グリップ**ボタンを押します。
- **ランダムな背景:** **Bボタン**（右コントローラー）を押すと、データセットからランダムな360度写真が読み込まれます。
- **デフォルトの背景:** **Aボタン**（右コントローラー）を押すと、デフォルトの背景に切り替わります。

## 技術とデータ

- **フレームワーク:** [A-Frame](https://aframe.io/)を使用して構築されています。
- **スカイボックス画像:** ランダムな背景画像は [vr-fukui.csv](https://code4fukui.github.io/vr-fukui/vr-fukui.csv) データセットから取得しています。
