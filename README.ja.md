# VRバンジージャンプ

このWebアプリケーションでは、オキュラスタッチコントローラーを使ってVRバンジージャンプのアニメーションを体験できます。実際のバンジージャンプに近い物理シミュレーションを実現し、背景画像の切り替えなども可能です。

## デモ
https://code4fukui.github.io/bungee-jumping/

## 機能
- 現実的なバンジージャンプのアニメーション
- オキュラスタッチコントローラーによる操作
- 背景画像の切り替え

## 必要環境
このプロジェクトはオキュラスVRヘッドセットとコントローラーが必要です。

## 使い方
1. 対応するVRブラウザでデモページを開きます。
2. オキュラスタッチコントローラーを使ってシーンを操作します。
3. トリガーやグリップボタンを押すとシミュレーションがリセットされます。
4. Bボタンを押すと背景画像が切り替わります。
