# VRバンジージャンプ

VRバンジージャンプを体験できるWebアプリケーションです。バンジージャンプのアニメーションを表示し、オキュラスタッチコントローラーを使って操作できます。

## デモ
https://code4fukui.github.io/bungee-jumping/

## 機能
- 現実に近いバンジージャンプのアニメーション表示
- オキュラスタッチコントローラーによる操作
- 背景画像の切り替え

## 必要環境
このプロジェクトはオキュラスVRヘッドセットとコントローラーが必要です。

## 使い方
1. 対応するVRブラウザでデモページを開きます。
2. オキュラスタッチコントローラーを使って、シーンと操作します。
3. トリガーやグリップボタンを押すとシミュレーションがリセットされます。
4. Bボタンを押すと背景画像が切り替わります。
