# Bungee Jumping VR

A VR bungee jumping simulation.

## Demo
https://code4fukui.github.io/bungee-jumping/

## Features
- Realistic bungee jumping simulation with physics calculations
- Animated sky background
- Countdown timer before jumping
- Reset functionality with hand controls

## Requirements
This project requires an Oculus VR headset and controllers to run.

## Usage
1. Open the webpage in a compatible VR browser.
2. Use the Oculus Touch controllers to interact with the scene.
3. Press the trigger or grip button to reset the simulation.
4. Press the A or B button to change the sky background.
